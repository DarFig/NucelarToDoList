# Nucelar ToDo List

## what is this?

This is a simple ToDo list application built using Electron.

## Requirements

    - NPM
    - Electron: npm install electron

## Package and Installer
    - Asar: npm install -g asar
    - Electron-builder: npm install -g electron-builder
and use: build -<option>
    - -w --windows
    - -l --linux
    - -m --mac
    - etc -h
